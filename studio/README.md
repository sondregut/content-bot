# Athlete Mindset Image Studio

Local web app for generating TikTok/Instagram carousel slides using the latest OpenAI image model.

## Setup

```bash
cd tiktok-content/studio
npm install
```

Create a `.env` file in `tiktok-content/studio`:

```bash
OPENAI_API_KEY=your_key_here
```

## Run

```bash
npm run dev
```

Open: http://localhost:4545

Images are saved to `tiktok-content/studio/output`.
